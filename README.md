# Supermarket Sales Analysis

## 📌 Project Overview
This project analyzes supermarket sales data to understand customer behavior, product performance, and sales trends.

## 📊 Dataset
The dataset contains daily sales records including:
- Product line
- Branch
- Customer type
- Gender
- Total sales

## ⚙️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🔍 Key Analysis Performed
- Data cleaning and preprocessing
- Top 3 product lines by sales
- Branch-wise average sales
- Customer type comparison
- Visualization (bar chart, box plot, histogram, heatmap)

## 📈 Key Insights
- Certain product lines generate higher revenue consistently
- Some branches outperform others in average sales
- Customer type impacts spending behavior

## 🚀 Conclusion
The analysis helps identify high-performing products and customer segments, which can support business decision-making.
